## Neutrino Utilities

**Neutrino Utilities is a collection of all of my tools I made using bash, python or some other language.**

Feel free to use any of these tools for yourself or modify them.
Don't forget to credit *me <3*

Please use the install.sh in tools/bash/ to install all the tools directly if you are unsure on how to use it.
Here is a quick description of each tool or script:

| Name | What it does | Potential usecase | Remarks |
|----------|----------|-----------|------|
| **Install** | Installs all the tools mentioned below into your system | If you don't know how to use the scripts or want them to be available at your user shell | Currently only working on Linux (Will write macOS Specific code "soon") |
| WinRB    | Reboots directly to Windows 10/11 from Linux| If you dual boot between Windows and Linux and need to switch between them a lot. | Only works on UEFI Systems |
| Netcheck | Constantly keeps pinging a server until it receives a stable ping. | Useful for places with unstable internet or power outages when you want to know when exactly internet comes back. | N/A |
